== Description ==
Do you want to manage notes for login and nonlogin users? 
Do it with WP-Frontend Notes.
Frontend notes enable you to store multiple notes on your website. This plugin will manage notes on the front end. It contains two shortcodes, one shows notes of all users, and the other shows current user notes (whether the plugin user is logged in or not ). It also contains edit, insert, and delete actions from which the user can easily edit their notes according to their preferences. Both users can able to see their details such as their name and image and their all inserted notes.All the notes are editable and easily accessible by the user.
